Contriboo is a python library for working with the github API and the git version control system. Based on it, you can quickly create services to track the productivity of IT company employees or repository statistics.

## Quickstart
---
Use 
clone the repository
```
git clone https://github.com/contriboo/contriboo-lib.git
```
and install deps with [`uv`](https://docs.astral.sh/uv/):
```sh
uv sync
```
